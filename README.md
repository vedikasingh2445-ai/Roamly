# 🏡 Wanderlust - Airbnb Clone

Wanderlust is a full-stack rental marketplace inspired by Airbnb that allows users to explore, create, edit, and manage property listings. The platform offers secure authentication, image uploads, reviews, and a seamless user experience for discovering unique stays.

## 🚀 Live Demo

🔗 **Deployment:** https://roamly-2v8t.onrender.com/listings

## ✨ Features

* User Authentication & Authorization
* Create, Edit, and Delete Listings
* Property Image Uploads with Cloudinary
* Add and Manage Reviews
* Responsive User Interface
* Secure Session Management
* MongoDB Database Integration
